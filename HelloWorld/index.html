Exemplos

Aqui, apresentaremos alguns métodos simples para realizar o famoso "Hello World" em Python. Lembre-se de que, embora a tarefa possa parecer simples, cada abordagem oferece insights valiosos sobre a sintaxe e os recursos da linguagem.

Método 1: Usando a Função print()

print("Hello World")

Este é um exemplo direto de como usar a função embutida print() para exibir a mensagem "Hello World" no console. A função print() é frequentemente usada para exibir informações para o usuário ou para depurar o código.

Método 2: Armazenando em Variável

mensagem = "Hello World"
print(mensagem)

Aqui, armazenamos a mensagem "Hello World" em uma variável chamada mensagem e depois a imprimimos usando a função print(). Isso ilustra como atribuir valores a variáveis em Python.

Método 3: Criando uma Função

def saudacao():
    return "Hello World"

mensagem = saudacao()
print(mensagem)

Neste método, definimos uma função chamada saudacao() que retorna a mensagem "Hello World". Em seguida, atribuímos o resultado da função à variável mensagem e a imprimimos. Isso nos dá uma visão da estrutura de definição e chamada de funções em Python.
